### Approach

- 自動化は効率化のためではなく、**自分にしかできないことに時間を使うため**
- AI に何を任せ、何を任せないかの**判断軸**を、現場経験から組み立てる
- ツールは**作りっぱなしにせず、組織で使われる形まで仕上げる**
- 思想と実装は分けない。**両方を一人で持つ**ことが強み
